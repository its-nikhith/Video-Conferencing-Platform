# Video-Conferencing-Platform

This project is a full-stack video conferencing platform built to support smooth online meetings with guests. It includes essential conferencing features such as video/audio calls, chat, screen sharing, and room management. Users can join as hosts or guests and collaborate in real time using an intuitive UI.  

* Features

1.Create & Join Meetings – Generate unique meeting rooms or join using a meeting ID.

2.Guest Access – Participants can join without signing up.

3.Real-Time Video & Audio – High-quality peer-to-peer communication using WebRTC.

4.Chatbox – Send instant messages to all participants during the meeting.

5.Screen Sharing – Share your screen with others for presentations or demos.

6.Mute/Unmute & Camera Toggle – Control your audio and video easily.

7.Responsive UI – Works smoothly on desktop and mobile devices.

*This project demonstrates how to build a modern real-time communication application from scratch using WebRTC and Socket.io. It can be used as a learning project, portfolio piece, or extended into a production-ready conferencing tool.
